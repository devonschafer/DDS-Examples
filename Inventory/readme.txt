Simple inventory program I built using DDS to test DDS.
Everything is built in python3 and tkinter.
Run "main.py" in terminal to launch the GUI.
"ui.dds" keeps some parameters for the GUI.
"dds.py" is the dds module.
"bench_stock.dds" is the historical inventory file.
Super simple, I built this because I am moving and need to re-organize/account for all of my parts.
Hope you enjoy this example.
